# xy

A Clojure library that wraps JTS.

[![Build Status](https://travis-ci.org/tetriscode/xy.svg?branch=master)](https://travis-ci.org/tetriscode/xy)

## Usage

```
=> (xy.geojson/parse (slurp "resources/sample.geojson"))
```

lein test
